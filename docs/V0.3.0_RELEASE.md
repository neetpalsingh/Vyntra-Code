# Vyntra Code v0.3.0 - Advanced Features Release

## 🎉 Major New Features

### 1. Multi-File Editing with Diff Preview ✨

**DiffView Component** - Professional diff viewer with split/unified modes

Features:
- ✅ Side-by-side diff comparison
- ✅ Unified diff mode
- ✅ Syntax highlighting in diffs
- ✅ Accept/Reject individual files
- ✅ Batch accept/reject all changes
- ✅ Status indicators (pending/accepted/rejected)
- ✅ Line numbers
- ✅ Expand/collapse file views

**Usage:**
```typescript
// Agent makes changes to multiple files
// User reviews in diff view
// Accept or reject each change
```

## 🚀 Usage Examples

### Multi-File Editing
1. Agent makes changes to multiple files
2. Click "Diffs" in navigation
3. Review each file's changes
4. Accept or reject individually
5. Or use "Accept All" / "Reject All"

### File Context Selection
1. Click "Files" in navigation
2. Browse workspace tree
3. Click "+" on files to add to context
4. Selected files shown in counter
5. Use in next AI query

### Agent Monitoring
1. Run agent task
2. Click "Agent" in navigation
3. Watch real-time progress
4. Expand steps for details
5. View tool calls and results

### RAG Search
1. Click "RAG Search" in navigation
2. Enter search query
3. View relevance scores
4. Click to expand code preview
5. Add results to context

### Export Conversation
1. Open conversation sidebar
2. Hover over conversation
3. Click download icon
4. .md file downloads automatically
